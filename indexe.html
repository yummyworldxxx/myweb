★★★⋆ 3.5_Mistake
「Ended up like oil noodles lol」
(Sapporo Miso Ramen Specialty Shop Keyaki Supervised — Spicy Miso Hot Pot by Mizkan)
I messed up the noodle-to-soup ratio (too many noodles), so it basically turned into *abura soba* (oil noodles). But I think it was still pretty good.
Because I messed up the cooking, I’ll rate it 3.5 (that’s on me). I’ll buy it again sometime and check the real taste then.
I’ll also see if it actually has that “chemical flavor” I thought I noticed before — and decide if it’s a 4.0 with that in mind. The mistake made the memory of the taste kinda fuzzy lol. But I remember it being tasty lol.
■Date eaten: 25/11/13
■How I ate it: 1 pack × 200ml water × only Chinese noodles ⇒ 3.5

「Spicy and Delicious Ramen」

As for that artificial taste, I’ll just have to try it again to be sure.
