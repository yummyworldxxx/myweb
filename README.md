## Rating System

```
5.0  God
4.5  Definitely want to eat again
4.0  Want to eat again
------ The 4.0 Wall
       Beyond the high bar of “pretty good” — the real question is:
       Do I actually want to eat it again?

3.5  Pretty good
3.0  Normal
2.0  Not tasty
1.0  Terrible
```

**Note:**
Even if something is “delicious,”
if it doesn’t make you want to eat it again, it’s rated **3.5**.
